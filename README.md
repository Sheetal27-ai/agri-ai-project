# 🌱 Smart Farming AI System

## 🚀 Overview

This project is an AI-based system that helps farmers decide the best crops to grow based on soil, weather, and profit analysis.

---

## 🧠 Features

* 📸 Image-based soil detection
* 🌦️ Weather-based analysis
* 🤖 Machine Learning crop prediction
* 💰 Profit-based crop ranking
* 🖥️ Interactive Streamlit UI

---

## 🛠️ Tech Stack

* Python
* Scikit-learn
* Streamlit
* OpenCV
* Pandas

---

## ⚙️ How to Run

```bash
git clone https://github.com/Sheetal27-ai/agri-ai-project.git
cd agri-ai-project
pip install -r requirements.txt
python -m streamlit run app.py
```
